## Overview

electron-babel-react-boilerplate

## Tools

* ES6
* React
* gulp
* browserify

## Getting Started

install electron

    npm install electron-prebuilt -g 

make package.json file

    npm init -y

set up the necessary files.

    bundler install --path vendor/bundle
    npm i
    bower i
    
and

run a electron

    npm run start
